View the interactive API documentation for Town Builder.

After starting the development server, visit:
- **Swagger UI**: http://127.0.0.1:5001/docs
- **ReDoc**: http://127.0.0.1:5001/redoc

The API documentation is automatically generated from FastAPI route definitions and includes:
- All available endpoints
- Request/response schemas
- Authentication requirements
- Try-it-out functionality

You can also view the OpenAPI schema at: http://127.0.0.1:5001/openapi.json
